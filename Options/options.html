Comment utiliser cette extension :

  1. Sélectionnez du texte sur n'importe quelle page Web.
  2. Cliquez avec le bouton droit pour afficher le menu contextuel et choisissez "Répondre".
  3. Ou utilisez le raccourci clavier ⌘Cmd + ⇧Maj + & (macOS) ou Ctrl+I (Windows/Linux) pour envoyer la requête directement à l'assistant.
  4. Si vous souhaitez modifier le raccourci pour déclencher l'envoi vers ChatGPT, vous pouvez aller dans chrome://extensions/shortcuts et cliquer sur l'icône de stylo pour modifier les raccourcis selon vos préférences.

Comment obtenir une clé API :

  1. Allez sur le site Web d'OpenAI et créez un compte.
  2. Accédez à la section API pour générer une nouvelle clé API.
  3. Copiez cette clé et collez-la dans le champ API Key des options de l'extension.

Paramétrage de l'extension :

  • Modèles d'IA : Choisissez parmi les modèles GPT-4, GPT-4 Turbo, et GPT-4o pour adapter la performance de l'extension à vos besoins.
  • Mode STRI : Activez ou désactivez la fonctionnalité de sélection automatique des réponses pour les quiz sur stri.net .
  • Notifications : Activez ou désactivez les notifications de l'extension pour chaque réponse de l'assistant.
